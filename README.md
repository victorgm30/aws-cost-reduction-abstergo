# AWS Cost Reduction Strategy – Abstergo Industries

Projeto desenvolvido no Bootcamp com foco em redução de custos para uma empresa farmacêutica que não utiliza computação em nuvem.

# Estratégia aplicada

O projeto foi estruturado em três etapas:

-> Virtualização da infraestrutura com Amazon EC2

-> Escalabilidade automática com EC2 Auto Scaling

-> Otimização inteligente de armazenamento com S3 Intelligent-Tiering

# Objetivo

Redução de custos operacionais, eliminação de infraestrutura física e implementação de modelo escalável baseado em nuvem.
